# Netflix Content Analysis Dashboard

## Project Overview
This project analyzes Netflix movies and TV shows using Power BI to uncover content trends, genre distribution, regional availability, and platform growth patterns.

## Tools Used
- Power BI
- Power Query
- DAX
- Google Sheets / Excel

## Key Features
- KPI Cards
- Genre Distribution Analysis
- Country-wise Content Analysis
- Movie vs TV Show Comparison
- Yearly Content Growth Trend
- Interactive Slicers and Filters

## Data Cleaning
- Handled missing values
- Removed duplicates
- Parsed date fields
- Split duration values
- Extracted primary genres and countries
- Created analytical helper columns

## Dashboard Insights
- Drama and Comedy dominate platform content
- Movies significantly outnumber TV Shows
- United States contributes the highest title count
- Content additions increased rapidly after 2015
